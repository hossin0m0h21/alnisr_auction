# ❓ الأسئلة الشائعة - ALNISR FAQ

## 🚀 البدء والتثبيت

### س: كيف أبدأ المشروع؟
**ج:**
```bash
git clone https://github.com/yourusername/alnisr-auction.git
cd alnisr-auction
npm install
npm start
```
ثم افتح `http://localhost:3000`

### س: هل أحتاج MongoDB؟
**ج:** نعم، المشروع يعتمد على MongoDB. استخدم:
- MongoDB Atlas (سحابي - الأفضل)
- MongoDB محلي للتطوير

### س: كيف أعدّل المتغيرات البيئية؟
**ج:** عدّل ملف `.env`:
```
MONGO_URI=your_connection_string
STRIPE_SECRET_KEY=your_key
PORT=3000
```

### س: هل يمكنني استخدام port مختلف؟
**ج:** نعم، عدّل في .env:
```
PORT=8080
```

## 💳 المدفوعات

### س: كيف أفعل Stripe؟
**ج:**
1. أنشئ حساب على [stripe.com](https://stripe.com)
2. احصل على Secret Key من Dashboard
3. أضفه في `.env`: `STRIPE_SECRET_KEY=sk_live_...`
4. سيعمل الدفع مباشرة

### س: هل أحتاج بطاقة بنكية لـ Stripe؟
**ج:** نعم، تحتاج بطاقة بنكية للحصول على حساب أساسي. أما في الاختبار يمكنك استخدام أرقام وهمية.

### س: كيف أختبر الدفع بدون فلوس حقيقية؟
**ج:** استخدم بطاقات الاختبار:
- `4242 4242 4242 4242` - دفع ناجح
- `4000 0000 0000 0002` - دفع مرفوض
- أي تاريخ وأي CVC

### س: هل أستطيع استخدام PayPal؟
**ج:** نعم، إذا أضفت PayPal Credentials في `.env`:
```
PAYPAL_CLIENT_ID=your_id
PAYPAL_SECRET=your_secret
```

### س: ماذا عن التحويل البنكي؟
**ج:** يتم يدويا:
1. الفائز يحول للحساب البنكي المعطى
2. الإدارة توثق التحويل
3. يُضغط على "Confirm Transfer" من الإدارة

## 📧 الإشعارات والبريد

### س: كيف أفعل البريد الإلكتروني؟
**ج:**
1. أنشئ Gmail account أو استخدم Sendgrid
2. فعّل "App Passwords" في Gmail
3. أضف في .env:
```
EMAIL_SERVICE=gmail
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-app-password
```

### س: لماذا رسائلي لا تصل؟
**ج:** تحقق من:
- المتغيرات في .env صحيحة
- App Password مفعل في Gmail
- البريد غير محجوب (شيك Spam)

### س: كيف أرسل SMS؟
**ج:**
1. أنشئ حساب على [twilio.com](https://twilio.com)
2. احصل على SID و Auth Token
3. أضف في .env:
```
TWILIO_ACCOUNT_SID=your_sid
TWILIO_AUTH_TOKEN=your_token
TWILIO_PHONE_NUMBER=+1234567890
```

### س: الـ SMS تكلفه؟
**ج:** نعم، Twilio بتحسب مبلغ صغير لكل SMS. لكن في البدايه الحساب الجديد يعطيك credit مجاني.

- قاعدة بيانات منفصلة

**ج:** نعم:
- Stripe يتعامل مع البطاقات (لا نحفظها)
- Webhooks موثوقة
- عدم الكشف عن الهوية

## 🚀 النشر

### س: أين أنشر المشروع؟
**ج:** خيارات جيدة:
- **Render** (الأسهل والأرخص)
- **Railway** (سريع وسهل)
- **Vercel** (للـ Frontend فقط)
- **AWS EC2** (للأداء العالي)

### س: كيف أنشر على Render؟
**ج:**
1. أنشئ حساب على render.com
2. ربط GitHub
3. اختر "New Web Service"
4. وصل المستودع
5. عدّل المتغيرات والأوامر
6. Deploy!

### س: كيف أنشر في Docker؟
**ج:**
```bash
docker build -t alnisr .
docker run -p 3000:3000 --env-file .env alnisr
```

### س: هل أحتاج نطاق خاص (Domain)؟
**ج:** بدون = `app-name.onrender.com`
مع = `myauction.com` (تحتاج شراء من GoDaddy مثلا)

## 🐛 المشاكل الشائعة

### خطأ: Cannot find module 'dotenv'
**الحل:**
```bash
npm install dotenv
```

### خطأ: MONGO_URI is required
**الحل:**
- تأكد من .env موجود
- `MONGO_URI` معرّف
- الاتصال صحيح

### المزاد لا يبدأ
**الحل:**
- تحقق من الوقت صحيح
- تحقق من الخادم يعمل
- رّجع الصفحة

### الدفع لا يعمل
**الحل:**
- تحقق Stripe Key صحيح
- جرب بطاقة اختبار
- تحقق من الـ logs

### الإشعارات لا تصل
**الحل:**
- تحقق Email متغيرات صحيحة
- تحقق من Spam folder
- جرب SMS بدلا من Email
